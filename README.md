# Gerenciador de Tarefas

Este é um projeto simples usando React para criar um gerenciador de tarefas, criado com Vite, utilizando React Router, Context API, useState, useEffect e localStorage

##Tecnologias usadas

- React
- Vite  
- React Router DOM  
- Context API  
- JavaScript (ES6+)  
- localStorage  
- CSS

## Como usar

Clone o repositório:

```bash
git clone https://github.com/usuario/nome-do-repositorio.git

Acesse o diretório:

cd nome-do-repositorio

npm install

Inicie o servidor local:

npm run dev

Acesse no navegador:

http://localhost:5173
